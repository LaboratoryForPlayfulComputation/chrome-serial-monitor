Serial Monitor

The single variable viewer is designed for you to be able to quickly work with a single number variable coming over the serial port of your micro:bit. This number will be streamed as a list in the Raw Stream view as well as in the form of a line graph over time. For example code to install on your micro:bit, see the Code Example section of this tool!
